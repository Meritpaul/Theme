# ModernShop - Premium E-commerce HTML Template

Version: 1.0.0

Thank you for purchasing ModernShop.

ModernShop is a modern, responsive, and feature-rich eCommerce HTML template built with Bootstrap 5. It is suitable for online stores, fashion shops, electronics stores, product showcases, and other eCommerce websites.

• Bootstrap 5 Framework
• Fully Responsive Design
• Modern User Interface

## INSTALLATION

1. Extract the downloaded package.
2. Upload all files to your hosting server.
3. Open index.html in your browser.
4. Customize the template according to your requirements.
