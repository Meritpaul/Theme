ModernShop Documentation

Version 1.0.0

Introduction

ModernShop is a modern, responsive, and easy-to-customize eCommerce HTML template built with Bootstrap 5. It is suitable for online stores, product showcases, fashion shops, electronics stores, and other shopping websites.

Features

  • Built with Bootstrap 5
  • Fully Responsive Layout
  • Modern and Clean Design
  • Cross-Browser Compatible
  • Easy Customization
  • Organized Code Structure
  • Fast Loading Performance
  • Mobile Friendly

Requirements

  • Modern Web Browser
  • Code Editor (VS Code Recommended)
  • Web Hosting Server (Optional for Deployment)

Installation

  1. Download and extract the template package.
  2. Upload all files to your hosting server.
  3. Open index.html in your browser.
  4. Customize content, images, CSS, and JavaScript as needed.

│   ├── images/
│   ├── fonts/
│   └── vendors/

└── documentation/
    └── index.html

assets/css/ – Contains all stylesheet files.
assets/js/ – Contains JavaScript files and functionality.
assets/images/ – Stores website images and graphics.
assets/fonts/ – Custom font files.
assets/vendors/ – Third-party libraries and plugins.
documentation/ – Template documentation files.

CSS Customization

Main stylesheet: 

assets/css/style.css

Edit this file to customize:

  • Colors
  • Typography
  • Spacing
  • Buttons
  • Layout Styles

Responsive CSS

Responsive styles are located in: 

assets/css/responsive.css

Use this file to modify tablet and mobile layouts.

JavaScript

Main JavaScript file: 

assets/js/main.js

Use this file to customize interactive components and site functionality.
